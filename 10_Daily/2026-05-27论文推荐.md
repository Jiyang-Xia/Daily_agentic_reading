**一句话总结**：推出了首个为张量程序优化提供原子级、单步监督和结构化思维链（CoT）推理的 Grounded 数据集 Step-TP。

**核心贡献/观点**：
- **解决单步决策难题**：传统数据集仅提供端到端优化对，导致 LLM 在庞大的组合优化空间中难以做出可靠的单步决策。

### TIAR: Trajectory-Informed Advantage Reweighting for LLM Abstention Learning
- **作者**：Muyu Pan, Shu Zhao, Nan Zhang, Philip Shin, Varun Parekh, Vijaykrishnan Narayanan, Rui Zhang
- **机构**：Pennsylvania State University
- **链接**：[arXiv](http://arxiv.org/abs/2605.25850v1) | [PDF](https://arxiv.org/pdf/2605.25850v1)
- **来源**：arXiv
- **笔记**：--

**一句话总结**：提出了一种在 GRPO 训练中利用多轨迹一致性动态重加权拒答奖励的算法，显著降低了大模型幻觉。

**核心贡献/观点**：
- **超越静态三值奖励**：传统拒答学习采用静态奖励，TIAR 通过将 GRPO 的多条采样轨迹作为天然的置信度指标，动态调整拒答优势（Advantage）。
- **探索知识边界**：利用轨迹一致性作为策略相对于查询的置信度指示器，鼓励模型在不确定时主动拒答。
- **幻觉抑制**：将研究重点从提升真诚度转向直接降低幻觉的拒答学习。

**关键结果**：在 AbstentionBench 评估中，TIAR 在 6 个类别中的 5 个取得了 SOTA 的拒答 F1 分数，在 17 个数据集上超越了静态基线，且完全保留了模型的基线准确率。

### MuCRASP: Multimodal Chain-of-thought Reasoning aware Structured Pruning
- **作者**：Aritra Dutta, Somak Aditya
- **机构**：Indian Institute of Technology Kharagpur
- **链接**：[arXiv](http://arxiv.org/abs/2605.25842v1) | [PDF](https://arxiv.org/pdf/2605.25842v1)
- **来源**：arXiv
- **笔记**：--

**一句话总结**：提出了一种专为多模态大模型设计的、能够保护思维链（CoT）推理能力的结构化剪枝框架。

**核心贡献/观点**：
- **识别 CoT 剪枝痛点**：指出传统剪枝会破坏 CoT 一致性，因为一致性高度依赖生成轨迹中的少数关键转移点（枢轴 Token），且未考虑跨模态激活分布的差异。
- **推理敏感型剪枝**：MuCRASP 专门针对非推理关键组件进行剪枝，同时保护跨模态对齐。
- **全局预算分配**：在全局参数预算下，根据层敏感度动态分配剪枝比例。

**关键结果**：在 Qwen2.5-VL-7B 上剪枝 30% 时，MuCRASP 在物理推理任务上的 LLM-as-a-Judge 得分为 8.87（对比最强基线 7.32），在剪枝高达 50% 时仍能保持极高的推理一致性与极低的困惑度退化。

### How Should LLMs Consume High-Quality Data? Optimal Data Scheduling via Quality-Aware Functional Scaling Laws
- **作者**：Zhitao Zhu, Xili Wang, Shizhe Wu, Jiawei Fu, Xiaoqing Liu
- **机构**：Tsinghua University, Huawei
- **链接**：[arXiv](http://arxiv.org/abs/2605.25698v1) | [PDF](https://arxiv.org/pdf/2605.25698v1)
- **来源**：arXiv
- **笔记**：--

**一句话总结**：通过将数据质量引入功能缩放定律，推导出联合数据质量与 Batch Size 调度的渐近闭式解，并提出了 Drop-Stable-Rampup 调度策略。

**核心贡献/观点**：
- **扩展 Scaling Laws**：首次将数据质量维度引入功能缩放定律（Functional Scaling Laws），为高质量数据的使用时机提供理论指导。
- **揭示高质量数据的双重角色**：
  - **噪声限制区（信号放大器）**：应降低 Batch Size，将更干净的数据转化为更多信号而不放大噪声。
  - **信号限制区（噪声抑制器）**：应在训练后期使用，以减少终端噪声。
- **提出 Drop-Stable-Rampup 策略**：在质量过渡时，先降低 Batch Size 并保持稳定以积累信号，随后逐渐增加以抑制终端噪声。

**关键结果**：在 15B MoE 模型和 108B Token 的中期训练（Midtraining）实验中，该策略相比 Cosine 衰减将 GSM8K 准确率提升了 +4.23，MATH 提升了 +2.80，全任务平均提升 +2.98。

### Multi-Agent Coordination Adaptation via Structure-Guided Orchestration
- **作者**：Haoran Li, Shulun Chen, Shaoyuan Sun, Hanchen Wang
- **机构**：Peking University
- **链接**：[arXiv](http://arxiv.org/abs/2605.25746v1) | [PDF](https://arxiv.org/pdf/2605.25746v1)
- **来源**：arXiv
- **笔记**：--

**一句话总结**：将多智能体协同建模为结构与编排联合分布上的后验概率推理，提出了兼顾稳定与适应性的 MACA 框架。

**核心贡献/观点**：
- **概率视角重塑协同**：指出传统方法要么结构过于僵化，要么编排过于混乱。MACA 将协同视为概率后验推理。
- **结构引导编排**：学习一个任务和预算约束下的结构先验（Prior），指导基于策略的动态编排，从而实现精细化控制。
- **抑制冗余交互**：通过联合自适应，使智能体之间的交互向最有效的路径收敛，大幅减少无用通信。

**关键结果**：在复杂任务基准上，MACA 相比自适应多智能体基线平均提升了 8.42% 的任务成功率，同时节省了 43.19% 的 Token 消耗。
